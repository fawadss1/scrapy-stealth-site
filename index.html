scrapy-stealth

Stealthy Crawling. Maximum Results.

A pluggable anti-bot and stealth framework for Scrapy — browser impersonation, proxy rotation, fingerprint cycling, and intelligent retry strategies for production-grade crawling.

Python 3.11+ Scrapy 2.12–2.x MIT License v…

Overview

scrapy-stealth extends Scrapy with browser impersonation, proxy rotation, fingerprint cycling, and intelligent retry strategies — designed for large-scale, production-grade crawling. It plugs in as a downloader middleware and lets you opt individual requests in or out of "stealth mode" using request.meta.

🧠 Why scrapy-stealth?

Scrapy is fast and powerful, but modern websites use advanced anti-bot protections such as TLS fingerprinting, browser behavior checks, and localized rate limits.

Browser-level impersonation

Deep emulation of modern TLS structures and HTTP/2 setups.

Smart Retry Strategy

Heuristics engineered to intercept temporary challenges without loops.

Proxy & Profile Rotation

Dynamically maps clean browser fingerprints matching routing states.

Advanced Anti-Bot Evasion

Defeats Cloudflare, Akamai and common static scripts out of the box.

⚙️ Integration & Setup

Option 1 — Environment Level (settings.py)

# 1. Enable the middleware
DOWNLOADER_MIDDLEWARES = {
    "scrapy_stealth.middlewares.StealthDownloaderMiddleware": 950,
}

# 2. (Optional) Route ALL requests through stealth automatically
STEALTH_ENABLED = True
STEALTH_DRIVER  = "turbo"   # "basic" (default), "turbo", or "browser"

# 3. (Optional) Proxy list for automatic rotation
STEALTH_PROXIES = [
    "http://proxy1:8080",
    "http://proxy2:8080",
    "http://user:pass@proxy3:8080",  # with authentication
    "socks5://proxy4:1080",
]

Option 2 — Isolated Spider Rules (custom_settings)

class MySpider(scrapy.Spider):
    name = "example"

    custom_settings = {
        "DOWNLOADER_MIDDLEWARES": {
            "scrapy_stealth.middlewares.StealthDownloaderMiddleware": 950,
        },
        "STEALTH_ENABLED": True,
        "STEALTH_DRIVER": "turbo",
        "STEALTH_PROXIES": [
            "http://proxy1:8080",
            "http://user:pass@proxy2:8080",
            "socks5://proxy3:1080",
        ],
    }
Proxies are validated at startup — invalid format or unsupported scheme raises ValueError immediately.

🚀 Quick Start

Option A — Per-request (stealth only on specific requests)

yield scrapy.Request(
    url="https://example.com",
    meta={"stealth": {}},
)

Option B — Global mode (stealth on every request automatically)

# No meta needed — all requests go through stealth
yield scrapy.Request(url="https://example.com")

# Opt out for a specific request
yield scrapy.Request(url="https://api.internal/health", meta={"stealth": False})

🔧 Global Configurations

Customise package-wide defaults via the shared config instance. All settings must be applied at module level, before the spider class.

# myspider.py
import scrapy
from scrapy_stealth.config import config

config.DEFAULT_ENGINE  = "stealth"      # "scrapy" (native) or "stealth"
config.DEFAULT_PROFILE = "chrome_147"   # fallback browser identity
config.DEFAULT_TIMEOUT = 30             # stealth request timeout in seconds
config.STEALTH_DRIVER  = "turbo"        # "basic", "turbo", or "browser"
config.HTTP2           = True
config.BLOCK_CODES    |= {407}          # expand blocked status codes
config.BLOCK_KEYWORDS.append("banned")
config.BROWSER_HEADLESS = True          # browser driver: headless mode
config.BROWSER_SETTLE_S = 4.0           # browser driver: seconds to wait for JS
config.BROWSER_EXECUTABLE_PATH = "/usr/bin/brave-browser"

class MySpider(scrapy.Spider):
    name = "example"
    ...

⚙️ Per-Request Config

All options are passed via request.meta["stealth"].

yield scrapy.Request(
    url,
    meta={
        "stealth": {
            "driver": "turbo",
            "profile": "chrome_147",
            "proxy": "http://user:pass@proxy:8080",
            "stealth_timeout": 60,
            "http2": True,
            "rotate_proxy": True,
            "rotate_profile": True,
        }
    },
)
Key Type Description
driver str "basic", "turbo", or "browser" — overrides config
profile str Browser profile (e.g. "chrome_147", "safari_ios_18_1_1")
proxy str Explicit proxy URL
stealth_timeout int Per-request timeout in seconds
http2 bool True = HTTP/2, False = HTTP/1.1
rotate_proxy bool Auto-pick a proxy from STEALTH_PROXIES
rotate_profile bool Auto-pick a random browser profile
headless bool Browser driver only: True = headless, False = visible
settle float Browser driver only: seconds to wait for JS (default 4.0)
snapshot bool Browser driver only: capture PNG to response.meta["snapshot_content"]

🖥️ Real Browser Integration Engine

Leverages Chrome DevTools connection logic directly to navigate dense, client-rendered JavaScript protections effortlessly without needing standard WebDriver setups.

Per-request Implementation

yield scrapy.Request(
    url,
    meta={
        "stealth": {
            "driver": "browser",
            "headless": False,   # harder to detect fingerprint
            "settle": 4.0,       # allow JS tasks to finalize
        }
    },
)

Heavy Cloudflare / Custom Binaries & Docker

from scrapy_stealth.config import config

# Increase settle time for heavy checks globally
config.STEALTH_DRIVER   = "browser"
config.BROWSER_SETTLE_S = 12.0

# Using a non-standard browser (e.g., Brave)
config.BROWSER_EXECUTABLE_PATH = "/usr/bin/brave-browser"

# Running in Docker as root (force no-sandbox)
config.BROWSER_NO_SANDBOX = True
Performance Notice: The browser engine is slower than basic/turbo (~5-15s execution profiles vs <2s). Isolate complex, protected URLs into this engine while routing base lookups via turbo.

📸 Screenshots

Capture a PNG screenshot of any page rendered by the browser driver and save it to disk.

Enable on the request

yield scrapy.Request(
    url,
    meta={
        "stealth": {
            "driver": "browser",
            "snapshot": True,
        }
    },
    callback=self.parse,
)

Auto-save with snapshot decorator

from scrapy_stealth.decorators import snapshot

class MySpider(scrapy.Spider):

    @snapshot
    def parse(self, response): ...

    @snapshot(path="stealth_shots/page.png")
    def parse(self, response): ...

    @snapshot(path=lambda r: r.url.split("/")[-1] + ".png")
    def parse(self, response): ...

🧩 Strategies

Proxy Rotation Handler

from scrapy_stealth.strategies.proxy import ProxyRotator
proxy_rotator = ProxyRotator([
    "http://proxy1:8080",
    "http://proxy2:8080"
])

yield scrapy.Request(
    url,
    meta={
        "stealth": {
            "proxy": proxy_rotator.get(),
        }
    },
)

Fingerprint Engine Rotation

from scrapy_stealth.strategies.fingerprint import ProfileRotator
fp = ProfileRotator()

yield scrapy.Request(
    url,
    meta={
        "stealth": {
            "profile": fp.get(),
        }
    },
)

Intelligent Retry Controller

from scrapy_stealth.strategies.retry import RetryHandler
retry = RetryHandler()

def parse(self, response):
    if retry.should_retry(response):
        yield retry.build(response.request)
        return

🛡️ Anti-Bot Detection

from scrapy_stealth.detectors.antibot import AntiBotDetector
detector = AntiBotDetector()

if detector.is_blocked(response):
    print("Block state confirmed.")

📊 Full Practical Implementation

import scrapy

class ExampleSpider(scrapy.Spider):
    name = "example"

    def start_requests(self):
        yield scrapy.Request(
            "https://example.com",
            meta={
                "stealth": {
                    "rotate_proxy": True,
                    "rotate_profile": True
                }
            },
        )

    def parse(self, response):
        yield {
            "title": response.css("title::text").get(),
            "url": response.url,
        }
